Important FAQ

What is the Document Object Model (DOM)?

DOM(Document Object Model) is a object like model made up of html elements.It represents the html elements as tree which helps to access,manipulate,add or remove the elements with js.

How do you select an element from the DOM?

We can select it by querySelector or by getElementById.
Example:
document.querySelector('selector');
document.getElementById('id');

What is event delegation in the context of the DOM, and why is it useful?

Its an process to add event listener to childnodes through their parentNode. The event listener is added to parentNode and it is applied to all the childNodes by default.

How do you manipulate an element's attributes and styles using the DOM?

The steps to manipulate an element,s attribute and style are:
Step 1:Access the element by getElementById or By querySelector.
Example:
const element=document.getElementById('id');
const element1=document.querySelector('id/class')
Step 2: Then we can just use getAttribute,setAttribute or removeAttribute to manipulate them.We can manipulate its style by simply using element.style.theStyleToChange.
Example:
element.style.color="yellow";
Or we can just add class by classList.add() and remove class by classList.remove()
Example:
element.classList.add();
element.classList.remove();
